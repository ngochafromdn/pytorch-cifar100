# Pytorch-cifar100


## Requirements

- python3.6
- pytorch1.6.0+cu101


## Usage

### enter directory
```bash
$ cd pytorch-cifar100
```

### dataset
 cifar-100-python

### train the model

```bash
# use gpu to train vgg16
$ python train.py -net vgg16 -gpu
```

Reason to choose vgg16: An example(which I trained first and took a long time), uniform architecture, achieving state-of-art performance

###  test the model
Test the model using test.py
```bash
$ python test.py -net vgg16 -weights checkpoint\vgg16\Tuesday_01_August_2023_16h_56m_27s\vgg16-60-regular.pth
```
